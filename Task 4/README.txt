- Данная программу реализована с помощью ThreadPool
- По умолчанию порт имеет 4 дока для загрузки / разгрузки (ThreadPool - 4)
- Максимальная вместимость порта 10000 контейнеров
- Максимальная вместимость корабля 100, загруженность корабля определяется случайным числом
- Коробки которые не смогли загрузиться, будет выведено соответствующее сообщение
- Всего в порту планируется разгрузиться 100 кораблей
- По итогу выполнения программы, в конце будет выведена информация о количестве контейнеров в порту
